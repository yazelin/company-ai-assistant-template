# Company AI Assistant Template CI Design

## Positioning

**Audience:** 給想先驗證公司內部知識庫問答的中小企業與工程團隊  
**Promise:** 先用 SQLite + keyword retrieval 做出可展示的內部文件 AI 助理，再逐步升級 RAG。  
**Category cue:** Docs / retrieval / ask API

## Visual Identity

- **Primary accent:** `#f59e0b`
- **Secondary accent:** `#14b8a6`
- **Background:** `#17120a`
- **Graphic motif:** `docs`
- **Style:** dark developer-tool landing page, technical grid, luminous product glyph, high-contrast CTA.

## Palette

- `#f59e0b`
- `#14b8a6`
- `#1c1917`
- `#fef3c7`

## Typography

- Display/UI: Inter
- Technical labels/code: JetBrains Mono
- Headline: large, tight tracking, high contrast
- CTA and labels: bold, concise, action-oriented

## Assets

- `assets/banner.svg`: Open Graph / README / hero banner
- `assets/logo.svg`: square product mark
- `index.html`: GitHub Pages CTA landing page

## CTA Strategy

Primary CTA collects email through FormSubmit and tags submission with `company-ai-assistant-template`.
Secondary CTA sends users to GitHub repo/template.
